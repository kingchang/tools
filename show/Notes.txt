Features:
- pager: terminal size detection
- view files: plug-in based (Git commits), encoding detection, binary files (archives, images), URL's, output transformation (syntax highlighting, syntax formatting, stop-if-slow, etc), case insensitive if not ambiguous
- compare files: Git support
- file monitoring: `tail`, Linux inotify
- list directories: `ls`
- find files: `find`
- search files: `ack-grep`
- view documentation: man pages, Perl, Python
- good defaults + optional commands

Requirements:
- Linux/Windows support
- fast: ~0 sec startup time, native-like fast
- extensible
