Features:
- pager: terminal size detection
- view files: plug-in based (Git commits), encoding detection, binary files (archives, images), URL's, output transformation (syntax highlighting, syntax formatting, stop-if-slow, etc)
- compare files: Git support
- file monitoring: `tail`, Linux inotify
- list directories: `ls`
- find files: `find`
- search files: `ack-grep`
- view documentation: man pages, Perl, Python
- good defaults + optional commands

Requirements:
- Linux support + Windows support (optional)
- fast: 0 sec startup time, native fast when replacing compiled code tools, native tools fallback (source-highlight vs Pygments)
- compilable with Cython or another tool for performance

Plan:
- library abstraction
- tests
- checks (Pylint, PEP8)
- profiling
- packaging: note down dependencies
- documentation

Dependencies:
- Run-time:
  - perl-doc: to view Perl documentation
  - pydoc: to view Python documentation
